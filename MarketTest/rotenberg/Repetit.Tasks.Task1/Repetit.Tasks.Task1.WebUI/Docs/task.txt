Допустим, есть интернет магазин, который продает некоторые товары. Пользователи могут оставлять заказы он-лайн или по телефону.
У заказа есть номер, дата заказа, ответственный менеджер(выбирается из таблицы в бд), адрес доставки, контактные данные, комментарии (текстовые поля) и список доставляемых товаров.
У доставляемого товара из списка есть тип товара и несколько дополнительных опций(выбираются из таблицы в бд).
Пример заказа:
Дата - 18.06.2013
Менеджер - Иванов Иван Иванович
Адрес доставки - Москва, Шарикоподшипниковская, 22
Контактные данные - 555-55-55
Комментарии - Звонить не позже 19-00
Список товаров:
        Товар 1:
                Тип - Велосипед
                Дополнительные опции:
                 1: Передний фонарь
                 2: Задний фонарь
                 3: Заднее крыло
        Товар 2:
                Тип - Роликовые коньки
                Дополнительные опции: Нет
        Товар 3:
                Тип - Фотоаппарат
                Дополнительные опции:
                 1: Сменный объектив
                 2: Штатив
Вам нужно составить схему таблиц в бд и реализовать форму редактирования заказа для сотрудников интернет магазина.
Форма должна давать возможность редактировать данные заказа(изменять ответственного менеджера, адрес и т.д.), добавлять/убирать товары в списке, добавлять/убирать дополнительные опции к товарам.
Для простоты будем считать что опции отдельно не продаются(это отдельная от товаров таблица), но для каждого типа товара есть разный список опций(нельзя добавить объектив к велосипеду).
